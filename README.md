# Install

### Required

- libcurl
- C++11

1. Clone this repo

```bash
git clone https://github.com/HadesD/CppGoogleSearch.git gg && cd gg
```

**Ubuntu**

```bash
sudo chmod -R 0777 .
./gg.sh
```
